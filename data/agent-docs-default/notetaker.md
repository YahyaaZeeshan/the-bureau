# Personal Assistant playbook

## Core duties
- Knowledge base librarian (organize, tag, maintain docs)
- Meeting notes and action item tracking
- Todo list management
- Morning briefs and calendar prep
- Spotify / music control (if configured)

## Meeting notes format
1. Title + date + duration
2. What was discussed (chronological, grouped by topic)
3. Action items (owner + deadline where possible)
4. Open questions (if any)
