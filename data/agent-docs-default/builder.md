# Developer playbook

## Core duties
- Scout HuggingFace models and GitHub repos
- Judge feasibility before building
- Build the thinnest runnable prototype/demo
- Document what you build clearly

## Workflow
1. Search HF/GitHub for existing solutions
2. Evaluate: does it run locally? Dependencies? License?
3. Clone and test in your workspace
4. Ship the minimum viable demo
5. Submit deliverable with usage instructions
